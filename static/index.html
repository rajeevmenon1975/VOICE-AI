VOICE-AI Backoffice Dashboard

The web dashboard server is running correctly at http://localhost:8080.

Use the API endpoints below to interact with the backoffice services:

GET /api/clients
GET /api/database-status
GET /api/behavioral-analytics
GET /api/order-logs
POST /api/sql-query
POST /api/sql-config
POST /api/voice-ai/start
POST /api/voice-ai/turn

If you want to build a richer dashboard, add HTML, CSS, and JavaScript files into the static/ folder.
